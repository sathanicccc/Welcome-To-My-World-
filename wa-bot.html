🤖 WHATSAPP BOT MAKING

1. Theory (എന്താണ് ബോട്ട്?)

WhatsApp-ൽ മെസ്സേജുകൾക്ക് ഓട്ടോമാറ്റിക് ആയി മറുപടി നൽകുന്ന ഒരു പ്രോഗ്രാം ആണ് ബോട്ട്. ഇതിനായി നാം ഉപയോഗിക്കുന്നത് Node.js എന്ന ഭാഷയാണ്.

2. Practical Steps:

  • GitHub Repo: ആദ്യം ഒരു ബോട്ട് സ്ക്രിപ്റ്റ് GitHub-ൽ നിന്ന് കോപ്പി (Fork) ചെയ്യുക.
  • Session ID: നിങ്ങളുടെ WhatsApp സ്കാൻ ചെയ്ത് കിട്ടുന്ന ഒരു കോഡ് ആണിത്. ഇതാണ് ബോട്ടിന് ജീവൻ നൽകുന്നത്.
  • Deployment: നിങ്ങളുടെ കമ്പ്യൂട്ടർ ഓഫ് ചെയ്താലും ബോട്ട് വർക്ക് ചെയ്യാൻ Heroku അല്ലെങ്കിൽ Koyeb പോലുള്ള സെർവറുകളിൽ ഇത് അപ്‌ലോഡ് ചെയ്യണം.

Required Codes: `npm start`, `pkg install nodejs`
